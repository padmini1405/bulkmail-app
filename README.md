## BulkMail Pro

    A modern bulk email sending web application built using the MERN Stack.
    This application allows users to send bulk emails, upload recipient lists, and track mail history with a clean and professional UI.

## Project Overview

    - BulkMail Pro is a web application developed using the MERN stack that helps users send bulk emails easily and efficiently.
    - Users can upload recipient email lists, compose messages, and send emails to multiple users at once.
    - The application also includes authentication features like signup and login for secure access.
    - A mail history page is provided to track previously sent emails and view their details.
    - The project is designed with a modern, responsive, and professional user interface for better user experience.

## Features

    - User Signup & Login Authentication
    - Send Bulk Emails
    - Upload Email Lists
    - Mail History Tracking
    - View Sent Mail Details
    - Responsive Professional UI
    - Error Validation Messages
    - Popup Modal for Viewing Mail Content
    - MongoDB Database Integration
